# Show user profile page

## Narrative

As a **signed-in user**,
I want **to view my profile with my name and account details**,
so that **I can confirm my information is correct and up to date**.

## Acceptance Criteria

**AC1:**
**Given** I am signed in,
**When** I navigate to my profile page,
**Then** I see my name, email address, and account creation date displayed.

**AC2:**
**Given** I am not signed in,
**When** I navigate directly to the profile URL,
**Then** I am redirected to the sign-in page.

## Implementation Notes

Profile data is returned by the `/api/me` endpoint. No additional backend work is needed.
