# Add user authentication

## Narrative

As a **new user**,
I want **to sign in with my email and password**,
so that **I can access my personal dashboard securely**.

## Acceptance Criteria

**AC1:**
**Given** I am on the sign-in page,
**When** I enter valid credentials and click Sign In,
**Then** I am redirected to my dashboard and see a welcome message.

**AC2 (integration):**
**Given** I am on the sign-in page,
**When** I submit the form with an unrecognised email,
**Then** I see an error message that does not reveal whether the email or password is wrong.

## Implementation Notes

Use the existing session-management library. Hashing is handled by the auth service.
