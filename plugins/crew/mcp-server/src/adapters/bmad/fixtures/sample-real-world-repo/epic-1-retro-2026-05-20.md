# Epic 1 Retrospective — 2026-05-20

## What went well

- The adapter registry seam (Story 3.1) made plugging in the BMad adapter
  straightforward. Keeping the interface narrow paid off immediately.
- Planning-discipline enforcement (Story 3.5) caught the missing-integration-AC
  pattern before any dev picked up the story.

  follow-up IDs (4-8b, 5-4b) were silently dropped.
- Missing Status lines caused the scan to halt completely instead of defaulting.
